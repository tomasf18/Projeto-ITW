FAQ'S

The Olimpics website was created to help you better know anything about the Olimpic Games. So, here you can find information about the athletes, competitions, coutries, games and modalities all organized using cards and charts.

There are two possibilities to search in the website. By using the search bar in the main page you can search everything about the Olimpics (athletes, competitions, coutries, games and modalities) and by selecting any option in the autocomplete box you will be direct to the details page of the option you'he choosen. The other way to search is using the search bar in the pages of athletes, modalities, coutries, games and modalities. Each of these search bars is usefull to search about especific information.

Of course. We really loved to work on it because we learned about the websites layout and how it works. Hope you enjoy using it!
